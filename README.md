### XynerraP2P

A minimal terminal-based chat over **pure WebRTC DataChannel** using Go and Pion.  
Peers connect directly by exchanging compressed SDP strings. No server involved.

---

### Usage

```bash
git clone https://github.com/catchmyass/XynerraP2P.git
cd XynerraP2P
go mod tidy
go run .
```
